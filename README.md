# TradeVaultX 🔥

> **Professional Trading Journal — by Milan**

TradeVaultX is a powerful desktop trading journal built for serious day traders. Track your trades, analyze your performance, and improve your edge — all in one clean, modern app.

## Features

### Dashboard
- Real-time P&L, Win Rate, Profit Factor, Average RR
- Account Balance chart with equity curve
- 7-day week overview
- Win Streak / Loss Streak tracker with flame indicator
- Monthly Goal progress bar
- Strategy & Symbol performance
- Best/Worst Day, Week, Month, Trade — all clickable
- Best & Worst Trading Sessions
- Buy Winrate & Sell Winrate
- Weekday Performance
- Best Trading Hours
- Period filter: Today / Week / Month / Year

## Installation

1. Download the latest `TradeVaultX Setup x.x.x.exe` from [Releases](../../releases)
2. Run the installer
3. Launch TradeVaultX from your Desktop

---

## Auto-Updates

TradeVaultX automatically checks for updates on launch. When a new version is available, it will download and install automatically.

---

## Tech Stack

- **Electron** — Desktop wrapper
- **HTML / CSS / JavaScript** — Single file app
- **Chart.js** — Charts and graphs
- **electron-updater** — Auto updates via GitHub Releases
- **localStorage** — Local data storage
